# `console.timeEnd(label)`

Ends a timer referent to the current label.

Status: `stable` <br />
Returns: `void`

### Parameters

| Name | Type | Description |
| ---- | ---- | ----------- |
| label | `string` | Identifier label for the timer |

### Example usage

```ts
console.timeEnd("test");
```
